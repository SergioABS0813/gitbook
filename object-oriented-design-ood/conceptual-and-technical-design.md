# Conceptual and Technical Design

Eliciting requirements: Proceso iterativo de investigación para recabar la mayor información posible sobre las necesidades del usuario.

Conceptual Design: Compone los primeros pensamientos sobre cómo los requerimientos serán cumplidos para el usuario. Sin ir más allá (parte técnica).

Technical Design: Se describen las funcionalidades de los componentes de manera técnica.

{% hint style="info" %}
Si los componentes o las conexiones entre ellos son imposibles de realizar en el Diseño Técnico, debemos ir nuevamente a la parte de Diseño Conceptual y reorganizar esa parte.
{% endhint %}
