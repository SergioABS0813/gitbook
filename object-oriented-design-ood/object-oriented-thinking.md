# Object Oriented Thinking

Object Oriented Thinking: Es una forma de representar las características clave de "algo" (perro, persona, pc).

Ayuda en programación ya que permite mantener el código organizado, flexible y reusable.

Es flexible porque permite realizar cambios sin afectar al resto de código.

Se puede reutilizar un objeto las veces que sean necesarias.

Los objetos físicos (reales) pueden ser representados en objetos de software
